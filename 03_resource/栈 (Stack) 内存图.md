
## 特点

**后进先出 (LIFO)** 跟叠盘子一样

### 栈内存图例

想象**栈内存**是一个只能从顶部存取数据的**箱子**。

- 声明一个变量时， `int a = 1;`，被**压入 (Push)** 到栈的顶部。
    
- 再声明一个变量， `int b = 2;`，会被**压入**到 `a` 的上方。
    
- 当一个函数执行完毕时，它所占用的内存空间会被从顶部**弹出 (Pop)**，只能按顺序从顶部拿走。
    

```
[     ] <-- 栈顶
[  b  ]
[  a  ]
[     ]
[     ] <-- 栈底
```

1. **压入 (Push) `a`**:
    
    ```
    [     ]
    [     ]
    [  a  ]
    [     ]
    ```
    
2. **压入 (Push) `b`**:
    
    ```
    [      ] <-- 栈顶
    [  b   ]
    [  a   ]
    ```
    
3. **弹出 (Pop) `b`**:
    
    ```
    [     ] <-- 栈顶
    [  a  ]
    ```
