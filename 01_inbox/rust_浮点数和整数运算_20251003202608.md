用 [[显式类型转换]] 来解决，最安全和最常见的做法是将 [[整数]] 转换为 [[浮点数]]，这样可以避免丢失精度

```rust
fn main() {
    let f = 2.4; // 类型推断为 f64
    let n = 20;  // 类型推断为 i32

    // 将整数 n 转换为 f64 类型，然后再与 f 相加
    // 'as f64' 就是显式类型转换
    let num = f + n as f64;

    // 我们可以打印结果来验证
    println!("f 的值是: {}", f);
    println!("n 的值是: {}", n);
    println!("num 的计算结果是: {}", num); // 结果会是 22.4
}
```
