# 追加偶数

```rust
fn generate_even_number_report() {
    let data = vec![1, 2, 3, 4, 5, 6, 7, 8];
    
    // 报告的初始状态是一个可变的空字符串
    let mut report = String::from("偶数报告:\n");

    // 遍历数据
    for num in data {
        if num % 2 == 0 {
            // 将偶数的信息追加到报告字符串中
            // to_string() 会把数字转换成字符串
            report.push_str(&num.to_string());
            report.push_str("\n"); // 加个换行符
        }
    }

    println!("{}", report);
}

// 调用函数
// generate_even_number_report();
```


# 不能被赋值因为是let

```rust
let mut s1 = String::from("A");
let mut s2 = String::from("B");

// my_ref 这个“绑定”是不可变的
let my_ref = &mut s1; 

// 错误！你不能让 my_ref 这个变量转而去指向 s2
// my_ref = &mut s2;
```


# 可以被赋值因为是let mut

```rust
let mut s1 = String::from("A");
let mut s2 = String::from("B");

// my_ref 这个“绑定”是可变的
let mut my_ref = &mut s1; 

// 完全可以！现在 my_ref 从指向 s1 转为指向 s2
my_ref = &mut s2; 

// 现在修改的是 s2
my_ref.push_str("!");
println!("{}", s2); // 打印 "B!"
```


# 通过 &mut 引用，去彻底替换“它指向的那个值

```rust
let mut s = String::from("Old value");

let my_ref = &mut s;

// 使用解引用 `*`，可以彻底替换掉 s 的值
*my_ref = String::from("New value");

println!("{}", s); // 打印 "New value"
```


# 四种特性和用途

### `mut` 关键字组合的四种情况总结

|语法|组合|含义|能做什么？|比喻|
|---|---|---|---|---|
|**1. `let my_ref = &s;`**|**不可变**绑定 + **不可变**引用|**标准的只读借用**|只能**读取** `s` 的数据。 不能修改 `s`。 不能让 `my_ref` 指向别处。|你有一张**只读**通行证，并且它被**粘死**在了你的手册第一页。|
|**2. `let mut my_ref = &s;`**|**可变**绑定 + **不可变**引用|**可重新赋值的只读借用**|只能**读取**当前指向的数据。 不能修改数据。 可以让 `my_ref` **换去指向**别的数据。|你有一张**只读**通行证，你可以随时把它换成另一张**只读**通行证。|
|**3. `let my_ref = &mut s;`**|**不可变**绑定 + **可变**引用|**专一的可变借用**|可以**读取和修改** `s` 的数据。 不能让 `my_ref` 指向别处。|你有一张**万能**通行证，但它被**粘死**在了你的手册第一页，只能用于那一页对应的房间。|
|**4. `let mut my_ref = &mut s;`**|**可变**绑定 + **可变**引用|**最灵活的可变借用**|可以**读取和修改**当前指向的数据。 可以让 `my_ref` **换去指向**别的数据，并修改那个新数据。|你有一张**万能**通行证，你可以随时把它换成另一张**万能**通行证，去改造别的房间。|

**核心要点：**

- **想修改数据？** `&` 后面必须带 `mut`。 (`&mut s`)
    
- **想重新赋值变量？** `let` 后面必须带 `mut`。 (`let mut my_ref`)
    

这两种能力是完全**独立**的，你可以根据实际需求，像搭积木一样自由组合它们。

这份总结应该可以作为你未来随时回顾的参考手册了！

# 将“可变变量”以“不可变引用”形式传递

```rust
// 这个函数只要求一个“不可变”的借用
fn inspect(s: &String) {
    println!("I am inspecting the string: {}", s);
}

fn main() {
    // 我们有一个“可变”的变量
    let mut my_word = String::from("Hello");

    // 我们把它的“不可变引用”传递给函数
    inspect(&my_word);

    // 在 inspect 调用结束后，我们依然可以修改它
    my_word.push_str(" World");
    println!("The word is now: {}", my_word);
}
```


# 将“可变引用”传递给需要“不可变引用”的形参

```rust
fn inspect(s: &String) { // 同样，只要求“不可变”
    println!("Inspecting via another ref: {}", s);
}

fn main() {
    let mut my_word = String::from("Mutable");
    
    // 我们先创建一个“可变引用”
    let mutable_ref = &mut my_word;

    // 然后直接把这个“可变引用”传给需要“不可变引用”的函数
    inspect(mutable_ref); // 这里会自动发生转换

    // 调用后，我们依然可以通过可变引用来修改数据
    mutable_ref.push_str(" Data");
    println!("Final word: {}", mutable_ref);
}
```
