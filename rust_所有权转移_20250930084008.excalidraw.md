# 成员变量都是标量类型

```rust
// 一个点，由两个 f64 浮点数组成
struct Point {
    x: f64, // 8 bytes
    y: f64, // 8 bytes
}

fn main() {
    let p = Point { x: 10.0, y: 20.0 };
    // 变量 p 的这 16 个字节数据，是作为一个整体直接存放在 main 函数的栈帧里的。
}
```


# 混合成员变量

```rust
struct User {
    id: u32,        // 4 bytes, 在栈上，位于结构体内部
    username: String, // 大约 24 bytes, 也在栈上，位于结构体内部, 但是...
}

fn main() {
    let user = User { id: 1, username: String::from("Rustacean") };
    // 'user' 变量本身在栈上。
    // 这块栈内存里包含了 id 的值 (1) 和 username 的三个信息（一个指向堆的指针、长度和容量）。
    // 而字符串的内容 "Rustacean" 本身，则是在堆（Heap）上分配的。
}
```
